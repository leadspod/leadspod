Getting Started Guide

Login

First login into our administrative dashboard at https://leadspod.com.

Add Email Account

Add your email accounts. If you activate your account it will also be warmed. So you don't need to activate them if you only want to use the account for sending.

Add Email Account Schedule

Go to the schedule tab to create the sending schedule for your email account. There is a tool in the help section that helps you to create a time schedule.

Add Email Server

To start sending an creating campaigns you need to order an email server first. You can have multiple accounts on one server.

Login Email Server

After you ordered your email server you will get the details via the mail. And you can login.

Contacts

Go to the contact section.

Create A Contact

In the contact section you can create an contact. You only need to create one contact.

Export Contacts

Now export the contacts.

Edit CSV File

You now have an CSV file that you can use to import accounts in bulk. Contact us if you need API access for most people the CSV file is the main method of importing contacts.

Create Segment

You need to create a segment before you can create a campaign.

Select Segment Filters

To fill your segment with contacts you need to create filters. The filters will be applied on all contacts and contacts that match will be added to your segment.

Create Email

After creating a segment you can create an email.

Edit Email Settings

In the advanced tab you can add the email account that will send the email in the From field.

Select Email Template

In the theme tab you can create your email template.

Edit Email In Builder

Create Campaign

After you created your emails you can create your campaign.
